- Il metodo createTextNode() in JavaScript serve per creare un nuovo nodo di testo nel Document Object Model (DOM) che può essere aggiunto a un elemento HTML esistente. A differenza di createElement(), che crea un nuovo tag HTML, createTextNode() crea solo il testo, senza alcun markup, attributi o elementi figli al suo interno. È utile per inserire del semplice testo in una pagina web, dopo averlo aggiunto a un altro elemento tramite metodi come appendChild().
